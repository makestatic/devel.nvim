# Devel.nvim

A dark Neovim colorscheme for devels

![Preview](./devel.png)
![Preview no frame](./devel-noframe.png)

## Installation (lazy.nvim)

```lua
{
  "makestatic/devel.nvim",
  lazy = false,
  priority = 1000,
  config = function()
    vim.cmd("colorscheme devel")
  end
}
```
